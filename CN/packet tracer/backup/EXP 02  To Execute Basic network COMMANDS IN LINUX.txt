EXP 02 AIM: To Execute Basic networking commands in Linux.


🧪 ✅ BASIC NETWORKING COMMANDS IN LINUX
🔹 1. IP Configuration
ifconfig
ip a
hostname -I

👉 Purpose:

View IP address, MAC address, network interfaces
🔹 2. Connectivity Testing
ping 8.8.8.8
ping google.com

👉 Purpose:

Check if network/internet is working
Uses ICMP
🔹 3. Routing Information
route
ip route

👉 Purpose:

Displays routing table
Shows default gateway
🔹 4. ARP Table (IP ↔ MAC Mapping)
arp -a
ip neigh

👉 Purpose:

Shows MAC address of connected devices
🔹 5. Network Connections & Ports
netstat
netstat -tuln
ss
ss -tuln

👉 Purpose:

Shows open ports and active connections
🔹 6. Trace Network Path
traceroute google.com

👉 Purpose:

Shows path (hops) to destination
🔹 7. DNS Information
nslookup google.com
dig google.com

👉 Purpose:

Resolves domain name to IP address
🔹 8. Interface Control
ip link show
ip link set eth0 up
ip link set eth0 down

👉 Purpose:

Enable/disable network interface
🔹 9. Hostname Information
hostname
hostnamectl

👉 Purpose:

Shows system name
🔹 10. Network Statistics
ifconfig -a
netstat -i

👉 Purpose:

Shows packet statistics
